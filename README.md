# security-audit-agent

AI-powered cybersecurity auditing agent for OpenCode, Claude Code, Codex, and Gemini CLI.

Audits your codebase for:
- **OWASP Top 10 (2021)** — A01 through A10
- **Mobile Security** — OWASP Mobile Top 10
- **Infrastructure & CI/CD** — Docker, cloud configs, pipeline security
- **LLM/AI Security** — prompt injection, training data exposure, model output validation

**Output:** Markdown reports + SARIF JSON for CI/CD integration.
**Platforms:** Web (React, Vue, Angular, Svelte), Mobile (Flutter, React Native, Kotlin, Swift), Backend (Node, Python, Go, Java), Cloud (AWS, GCP, Azure).
